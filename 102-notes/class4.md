# Class4-Notes Wireframe

Wireframes are like a brainstorm process placed on paper in order to create a plan for future website build.

HTML 'marksup' giving more details when transferring your framework version to the HTML code.

Semantics are generally accepted concepts or 'actions' in coding.

## Questions

1. What is HTML and why do we use it?
2. What are the 3 main parts of an HTML element?
3. What is it called when you give an element extra information?
4. What is a semantic element?

## Answers

1. HTML stands for *hyper text markup language*, helps identify the meaning and structure of web content.
2. The 3 main parts of an HTML elements are:

   + The opening tag (<>)
   + Content
   + The closing tag (</>)

3. Giving an element extra information is called an *attribute*

4. Semantics give meaning interpreted as action to a point to a piece of code. Think of purpose.
